# 69. x 的平方根

https://leetcode-cn.com/problems/sqrtx/

## 题目描述

实现 `int sqrt(int x)` 函数。

计算并返回 `x` 的平方根，其中 `x` 是非负整数。

由于返回类型是整数，结果只保留整数的部分，小数部分将被舍去。

**示例 1:**

```
输入: 4
输出: 2
```


**示例 2:**

```
输入: 8
输出: 2
说明: 8 的平方根是 2.82842..., 
     由于返回类型是整数，小数部分将被舍去。
```



## 思路

二分查找：

具体看代码，循环结束后 `low` 会指向比结果大 1 的数，最后返回时减去 1 即可。注意在计算平方的时候要防止溢出。



## 代码

```java
class Solution {
    public int mySqrt(int x) {
        int low = 1, high = x;
        while (low < high) {
            int mid = low + (high - low) / 2;
            // 防止溢出，这里使用除法
            if (mid == x / mid) {
                return mid;
            } else if (mid > x / mid) {
                high = mid - 1;
            } else {
                // 如果low发生了变化，循环过后会指向结果的下一个元素
                low = mid + 1;
            }
        }
        // 处理low一直没变的情况
        if (low <= x / low) {
            return low;
        }
        return low - 1;
    }
}
```
